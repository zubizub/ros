---------------------------------------------------------------------------------------------
-------------Основные команды gulp-----------------------------------------------------------

//основные
gulp watch - начать отслеживание изменений
gulp prod - скомпилировать js и css минимизированно (для продакшена). Сделать после завершения всех работ по css и js
gulp build - скомпилировать js и css НЕ минимизированно (для разработки)
